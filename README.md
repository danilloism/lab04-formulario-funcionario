## Como Executar com Docker

### Executar todos os exercícios

```bash
docker-compose up -d
```

### Acessar cada exercício

- **Exercício 1**: http://localhost:9001
- **Exercício 2**: http://localhost:9002
- **Exercício 3**: http://localhost:9003
- **Exercício 4**: http://localhost:9004

### Parar os containers

```bash
docker-compose down
```
